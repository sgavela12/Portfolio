Activate

Descripción

Activate es una aplicación web para monitorizar y gestionar hábitos, desarrollada con Spring Boot y Thymeleaf. Utiliza JPA y Hibernate para crear y gestionar dinámicamente la base de datos en MySQL, lo que permite que las tablas se generen automáticamente según las clases del modelo. Además, incluye un sistema de gestión de usuarios y contraseñas para una experiencia segura y personalizada.
